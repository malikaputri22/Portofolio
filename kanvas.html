Your browser does not support the HTML5 canvas tag.

Langkah-Langkah Membuat Canvas dengan HTML

  1. Gunakan tag <canvas> untuk membuat area kanvas di HTML.
  2. Tambahkan atribut width dan height untuk menentukan ukuran kanvas.
  3. Gunakan JavaScript untuk mengakses elemen kanvas dengan getContext('2d').
  4. Gunakan fungsi drawImage() untuk menggambar gambar ke dalam kanvas.
        
          <canvas id="kanvas" width="220" height="277"></canvas>

          <script>
            var c = document.getElementById("kanvas");
            var ctx = c.getContext("2d");
            var img = document.getElementById("gambar");
            ctx.drawImage(img, 0, 0, c.width, c.height);
          </script>
