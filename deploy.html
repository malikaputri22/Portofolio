Tutorial: Cara Deploy HTML ke GitHub Pages

Berikut adalah langkah-langkah untuk mendeply sebuah website statis berbasis HTML ke GitHub Pages.

Langkah-langkah:

  1. Buat akun GitHub jika Anda belum memilikinya, dan login ke akun Anda di GitHub.
  2. Buat repository baru di GitHub dengan nama repository, misalnya my-website.
  3. Upload file HTML dan asset website Anda (seperti gambar, CSS, dan JavaScript) ke repository tersebut.
  4. Jika menggunakan Git untuk upload, gunakan perintah berikut di terminal:
    5.           
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/username/my-website.git
git push -u origin main
  5. Setelah file di-upload ke GitHub, buka pengaturan repository (tab Settings).
  6. Scroll ke bawah sampai menemukan opsi GitHub Pages di bagian Code and Automation.
  7. Di bagian Source, pilih branch main dan direktori /root (atau lokasi lain jika Anda menggunakan subfolder).
  8. Setelah itu, GitHub Pages akan otomatis aktif, dan Anda akan diberikan link untuk mengakses website Anda. Biasanya link akan terlihat seperti ini:
    9.           
https://username.github.io/my-website/

Contoh Project HTML Sederhana:

Buat file HTML sederhana seperti ini:

        



  
  
  My Simple Website


  
Hello, World!

  
This is my first website deployed on GitHub Pages.

Selamat! Anda berhasil deploy website Anda ke GitHub Pages!
